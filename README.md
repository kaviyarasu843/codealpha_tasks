# codealpha_tasks
Hangman Game in Python

Project Description

This project is a simple text-based Hangman Game developed using Python. The player tries to guess a randomly selected word one letter at a time.

Features

- Random word selection
- Letter-by-letter guessing
- Limited incorrect attempts
- Win/Lose conditions
- Console-based interface

Concepts Used

- Random Module
- Loops
- Conditional Statements
- Strings
- Lists

Language Used

- Python
